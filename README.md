## 📝 규칙 0: 커밋 컨벤션

### 🎯 기본 커밋 컨벤션
```
solve: 문제번호 - 문제 제목
```

### 📁 파일명(클래스 이름) 컨벤션
```
Q_문제번호_티어_소요시간
```

### 📌 커밋 메시지 예시
- `solve: 1000 - A+B`
- `solve: 2798 - 블랙잭`

### 📌 파일명 예시
- `Q_1001_B5_104ms`
- `Q_2675_B2_92ms`

### 🔧 기타 컨벤션
- `refactor: 문제번호 - 개선내용` (코드 리팩토링)
- `fix: 문제번호 - 수정내용` (버그 수정)
- `docs: 내용` (문서 작성/수정)
- `chore: 내용` (폴더 구조 등 기타 작업)
